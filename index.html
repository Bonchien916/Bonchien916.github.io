Linux入门练习

Posted on 2018-07-04

通过命令行重启Linux

#reboot

快速删除光标前/后的内容

快捷键 前：ctrl+u 后 ctrl+k

如何删除/temp目录下所有A开头的

#rm -f /temp/A*

备份/etc/passwd 到 /temp目录

#cp /etc/passwd /temp/

如何查看系统中最后创建的3个用户

用户创建信息在/etc/passwd中

#tail -3 /etc/passwd

统计当前有多少用户

#wc -l /etc/passwd

如何创建/temp/text.conf文件

#touch /temp/text.conf

通过vim打开/temp/text.conf

#vim /temp/text.conf

如何查看/etc/passwd的头三行和尾三行

#head -3 /etc/passwd

#tail -3 /etc/passwd

如何一次性创建出/text/1/2/3/4

#mkdir -p /text/1/2/3/4

快速到当前账户的家目录

#cd ~

#cd

查看/etc所占磁盘空间

#du -sh /etc

删除/tmp下所有文件

#rm -rf /tmp/*

尝试启动apache服务，并检查是否启动成功

#service httpd start

#ps -ef | grep httpd

杀死apache进程

#killall httpd

运维入门第三天

Posted on 2018-07-03

ps 用于查看服务器进程
语法：ps 选项

例：ps -ef | grep xxx
选项：
-e 列出全部
-f 显示全部列（全字段）
字段解释：

UID PID PPID C STIME TTY CMD
执行该进程的用户ID 进程ID 该进程的调用者ID CPU占用率 启动时间 终端设备 该进程名称或路径

top 可以实时动态的查看系统运行情况
语法：top 选项

字段解释：
| PID | USER| PR | VIRT | RES | SHR |S |
| :——:| :——: | :——:| :——: | :——: | :——: |:——: |
| 进程ID | 该进程用户| 优先级 | 虚拟内存 | 常驻内存 | 共享内存 |状态 |
运行时可以使用快捷键
M：将结果按照内存（MEN）从高到低显示
P：CPU使用率从高到低
1：是否展示CPU详细信息

du 查看目录真实大小
语法：du 选项 文件

选项：
-s：summaries只显示汇总大小
-h：高可读性显示

find 查找文件
语法：find 路径范围 选项 选项的值

选项：
-name：按照文档名称模糊搜索
-type：文档类型 “f”表示文件 “d”表示文件夹
例：find /etc -name *.conf 表示搜索etc目录下后缀为conf的文件

service 启动服务
语法：service服务 start/stop/restart

例：service mysql start 启动mysql服务
可以通过ps -ef | grep mysql 来查看服务是否启动成功

kill 杀死进程
语法：kill 进程ID

这样有时候会麻烦，因为首先要知道知道进程ID，所以还有下面一种命令
语法：killall 进程名称

ifconfig 用于操作网卡相关指令

inet addr是网卡的ip地址

reboot 重启linux

shutdown 关机（慎用）
语法：shutdown -h now（root用户使用） 或者shutdown -h 15:30 （具体时间）

那么如何取消关机
在centos7.X之前：ctrl+c
在centos7.X之后：shutdown -c

除了shutdown，关机指令还有init 0 、halt、poweroff

uptime 输出持续在线时间

参数说明：系统当前时间 已运行时间 总连接数 系统平均负载

uname 计算机系统相关信息
语法： uname 选项

例：uname -a （获取完整信息）
参数说明：
类型 主机名 内核版本 发布时间 开源计划

netstat 查看网络连接状态
一般用法：netstat -tnlp

选项说明：
-t：只列出tcp协议的连接
-n：将地址从字母组合转化成ip、协议转化成ip
-l：state状态列中值为LISTEN（监听）的连接
-p：显示对应的PID和进程名称

man 手册
语法：man 命令

运维入门第二天

Posted on 2018-06-29

df 查看磁盘空间
语法：df -h

free 查看内存使用情况
语法：free -m(以MB为单位查看)

查了下网上的资料，发现也有这样的情况

解释：

total：表示物理内存总量。

used：表示总计分配给缓存（包含buffers 与cache ）使用的数量，但其中可能部分缓存并未实际使用。

free：未被分配的内存。

shared：共享内存。

buffers：系统分配但未被使用的buffers数量。

cached：系统分配但未被使用的cache数量。

buffers/cached中的total、used、free表示如下：

used：实际使用的buffers与cache总量，也是实际使用的内存总量。

free：未被使用的buffers与cache和未被分配的内存之和，这就是系统当前实际可用内存。

total(Mem) = used(Mem) + free(Mem)

total(Mem) = used(buffers/cache) + free(buffers/cache)

used(Mem) = buffers(Mem) + cached(Mem) + used(buffers/cache)

free(buffers/cache) = buffers(Mem) + cached(Mem) + free(Mem)

head/tail 查看文件前/末 n 行（默认10行）
语法：head/tail -n 文件名称

这里tail有第二种语法
tail -f 文件路径查看文件的动态变化
注意：不能手动打开

wc 统计文件内容信息
语法：wc -lwc 需要统计的文件内容

-l：lines行数
-w：words单词数（依据空格来判断数量）
-c：bytes字节数

date 查看日期 CST表示中国
语法：date 选项

例： date +%F(等价于+%Y-%m-%d)

还可以查看之前或者之后的时间。比如这样的语法date -d “-1 day” “+%F”,表示前一天

cal 查看日历
语法：cal 选项


cal -y 2018输出2018年的日历

（重要）管道符：| 通常用来“过滤”、”扩展”、”特殊”

管道前面的输出就是管道后面的输入

grep：主要用于过滤

运维入门第一天

Posted on 2018-06-28

1、认识Linux基本目录

bin 二进制文件（几乎所有用户命令），皆可运行
boot Linux启动时用到的一些文件
dev 外接设备，需要挂载使用，常用的是挂载光驱mount /dev/cdrom /mnt
lib 系统使用的函数库，比较重要的目录为/lib/modules
lost+found 系统异常产生错误时，会将一些遗失的片段放置于此目录下
mnt 光盘默认挂载点
srv 服务启动之后需要访问的数据目录，如www服务需要访问的网页数据存放在/srv/www内
etc 配置文件
home 除了root用户以外的其他用户的home目录
proc Linux运行时的进程
root root用户的Home
sbin super二进制文件，可运行，前提是有super权限的用户
temp 系统运行时产生的临时文件，用完会自动清理
usr 用户自己安装的软件
var 放置系统执行过程中经常变化的文件，如随时更改的程序/系统的日志文件/var/log，/var/log/message

2、Linux基本命令

Linux通用格式
指令主体 【选项】 【操作对象】
主体只有一个，选项可以多个，操作对象可以多个

指令
ls(list) 当前目录下所有文件、文件夹的名称

ls 路径 指定路径下的所有文件、文件夹的名称
相对路径：相对要有一个参照物（一般时当前目录下）
绝对路径：根目录开始

语法：ls 选项 路径
ls -l /root
ls -la /root
ls -lha /root
-l : 表示详细显示
-a : 显示所有文件/文件夹，包括隐藏的
-h : 高可读性


pwd 打印当前路径的绝对路径

cd 切换当前目录

语法：cd 路径(绝对路径)
cd /usr/local

常用：cd .. 回到上级目录
cd / 回到根目录
cd ~ 当前用户的home目录

mkdir 创建目录
语法一：mkdir 路径（可以是绝对、相对）
语法二：mkdir -p 绝对路径（此语法可用来一次性创建多层不存在的目录）
当前目录下直接mkdir xxx文件夹，像创建非当前目录的需要mkdir /../../xxx(绝对路径)，然后可以通过ls命令来验证是否创建了这个文件夹，简单不多说。
注意 颜色说明（用的Xshell）：
白色：表示普通文件
蓝色：表示目录
绿色：表示可执行文件
红色：表示压缩文件
浅蓝色：链接文件
红色闪烁：表示链接的文件有问题
黄色：表示设备文件
灰色：表示其他文件

touch 创建文件
语法：touch 绝对路径/文件名称（和mkdir类似）

cp 复制文件/文件夹
语法：cd 被复制的文件的绝对路径/文件名称.后缀 被复制绝对路径
文件夹复制时需要加选项 -r(递归复制)
例：
cd -r ./xxxx/ /home/xxxx

mv 移动、剪切
语法：mv 需要移动文档路径 需要保存的位置
注：重命名也是mv命令

rm 删除
语法：rm 选项 需要删除的文档路径
rm -f xxx 表示取消是否删除，直接删除
如果想删除 整个文件夹包括其中的文件 rm -rf xxx
有时候用到 通配符 ，比如 rm -f xxx**表示删除xxx开头的文件

vim 文本编辑
语法：vim 文件路径
编辑时v进入编辑，需要保存并退出:wq 直接退出:q

cat 直接打开一个文件、合并文件
语法一：cat 文件路径
与vim的区别就是 不需要:q退出
语法二：cat 待合并文件路径1 待合并文件路径2 >合并之后的文件路径
注：这里的>表示输出重定向，意思是只合并，不做后续操作，如果有后续操作用>>
