2018-06-28

1、认识Linux基本目录

bin 二进制文件（几乎所有用户命令），皆可运行
boot Linux启动时用到的一些文件
dev 外接设备，需要挂载使用，常用的是挂载光驱mount /dev/cdrom /mnt
lib 系统使用的函数库，比较重要的目录为/lib/modules
lost+found 系统异常产生错误时，会将一些遗失的片段放置于此目录下
mnt 光盘默认挂载点
srv 服务启动之后需要访问的数据目录，如www服务需要访问的网页数据存放在/srv/www内
etc 配置文件
home 除了root用户以外的其他用户的home目录
proc Linux运行时的进程
root root用户的Home
sbin super二进制文件，可运行，前提是有super权限的用户
temp 系统运行时产生的临时文件，用完会自动清理
usr 用户自己安装的软件
var 放置系统执行过程中经常变化的文件，如随时更改的程序/系统的日志文件/var/log，/var/log/message
